# dextermiller.com

Personal portfolio and resume site for Dexter Miller — IT Manager & Automation Engineer at Applied Technology Group in Fort Wayne, IN.

## About

This site serves as a professional portfolio showcasing:

- **Work experience** — 6+ years at ATG progressing from intern to Tier 1 Manager
- **Automation projects** — Rewst workflows handling 60–100 tickets/week
- **Data visualization** — Grafana dashboards for operational metrics
- **AI game suite** — Playable Tic-Tac-Toe and Connect Four using MinMax and Alpha-Beta pruning algorithms (Indiana Tech senior project)

## Tech Stack

- Static HTML5, CSS3, and vanilla JavaScript — no build tools required
- Hosted on GitHub Pages with custom domain (`dextermiller.com`)
- Responsive design with dark mode support

## Project Structure

```
├── index.html          # Home / About page
├── project.html        # Projects showcase & AI games
├── sitemap.xml         # XML sitemap for search engines
├── robots.txt          # Crawler directives
├── css/
│   ├── main.css        # Primary stylesheet (includes dark mode)
│   ├── tictactoe.css   # Tic-Tac-Toe game styles
│   └── connect.css     # Connect Four game styles
├── js/
│   ├── project.js      # Tic-Tac-Toe MinMax engine
│   ├── connect.js      # Connect Four Alpha-Beta engine
│   └── theme.js        # Dark mode toggle
├── images/             # Profile photo, campus photo, favicon
└── files/              # Algorithm diagrams referenced on Projects page
```
